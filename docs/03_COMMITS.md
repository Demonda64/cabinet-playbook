# Conventions de commits (solo)

Format :

- `chore:` maintenance / structure / doc
- `docs:` documentation
- `feat:` ajout
- `fix:` correction
- `refactor:` refacto sans changement fonctionnel

Exemples :

- `chore: init repository structure`
- `docs: add cabinet rules and mapping`
- `feat: add consulting checklist for discovery`
